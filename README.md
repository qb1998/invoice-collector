# 发票收集器 (Invoice Collector)

一个独立的发票自动收集平台，只需输入邮箱地址和 IMAP 授权码，即可自动从邮箱中收集全部发票及报销相关文件，提取关键信息并生成统计表格。

无需依赖任何外部服务，所有数据本地处理，隐私安全。

## 功能特性

- **一键收集**：输入邮箱+授权码+日期范围，自动抓取所有邮件附件
- **智能提取**：自动解析 PDF 发票，提取发票号码、日期、金额、买卖方等关键字段
- **自动分类**：按餐饮、住宿、交通、门票、商超等类别自动归类
- **辅助文件识别**：自动识别行程单、酒店确认单、登机牌等报销辅助文件
- **深度检查**：扫描无附件邮件正文中的发票下载链接（税务局、电子发票平台等）
- **发票去重**：相同发票号自动标注重复，红色高亮提示
- **分类汇总**：按类别统计金额、占比，自动生成汇总表
- **关联排列**：相关文件（如酒店水单与对应发票）自动相邻排列
- **文件打包下载**：一键下载 Excel 表格 + 全部文件 ZIP 包
- **自动重命名**：发票文件按 `序号-类别-日期-金额` 格式自动重命名
- **多邮箱支持**：163、QQ、Gmail、Outlook 等主流邮箱

## 快速开始

### macOS / Linux

```bash
./run.sh
```

### Windows

```bat
run.bat
```

首次运行会自动创建虚拟环境并安装依赖。启动后在浏览器打开 http://localhost:8000

### 手动启动

```bash
# 1. 创建虚拟环境
python3 -m venv .venv

# 2. 安装依赖
.venv/bin/pip install -r requirements.txt

# 3. 运行
.venv/bin/python app.py
```

## 使用方法

1. 在邮箱设置中开启 IMAP 服务并获取授权码
2. 浏览器打开 http://localhost:8000
3. 填写邮箱地址、授权码、日期范围
4. 点击「开始收集发票」
5. 等待收集完成，下载 Excel 表格或 ZIP 文件包

## 技术栈

| 组件 | 技术 |
|------|------|
| 后端框架 | FastAPI + Uvicorn |
| 前端 | 原生 HTML / CSS / JavaScript |
| PDF 解析 | pdfplumber |
| Excel 生成 | openpyxl |
| 邮件协议 | IMAP (imaplib) |

## 隐私说明

- 所有邮箱授权码和发票数据均在本地处理，不上传到任何服务器
- 授权码仅用于当前会话的 IMAP 连接，不持久化存储
- 收集的发票文件保存在本地临时目录中
